A simple "Guess the Number" game in **Rust**
## How to play

- The computer picks a number between 1 and 20
- You have 5 attempts
- After each attempt, the program will say "Too big!" or "Too small!"

## How to run
#1. Download the project(just paste it in terminal):
git clone https://github.com/joegoldberg110010100-it/guessing-game.git   
cd guessing-game

# 2. Launch and play!
cargo run
